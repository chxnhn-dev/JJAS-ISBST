# JJAS-ISBST Inventory & POS System

## Overview

JJAS-ISBST is a Windows Forms Inventory and Point of Sale (POS) system developed using **VB.NET** and **SQL Server**.

## Features

* Product management
* Inventory tracking
* Supplier management
* POS transaction processing
* Audit trail
* Sales reporting

## Technologies

* VB.NET (Windows Forms)
* SQL Server
* ADO.NET

## How to Run

1. Open `JJAS-ISBST.sln` in Visual Studio
2. Restore the database from the `DB` folder
3. Update the connection string in `App.config`
4. Run the project
